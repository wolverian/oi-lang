Oi the language

Hello and welcome! Oi is a smol proggin language. It doesn't understand much yet, so please be gentle. It's mostly modelled after io language, so you can get an idea of what Oi is like from reading the io language guide.

If you display is wide enough, you can view some things side-by-side. Try it!
